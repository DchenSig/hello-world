Three Kingdoms Warrior Game
Descriptions: The game is based on the Chinese historical period of three kingdoms. The game is similar to a chessboard game in which the users can control different types of warrior figures ¨C infantry, cavalry, archers and sorcerers ¨C to combat with other player or a computer AI. Each type of warrior figure has different attack abilities, move ranges and spell abilities. The goal of the game is to eliminate all the warriors of the other player. The user will play against a computer AI. The game requires the users to strategize their moves in each turn to achieve victory.

How to run the project: the user should run __init__.py to start the game

Libraries: pygame

Short cut commands: None
